## Laptop Decision Dashboard (Excel ETL & Data Modeling)
# Overview
This project is a decision-support dashboard designed to help users rank laptops based on their preferences. It uses Excel’s ETL capabilities to extract and transform web data, combined with advanced data modeling and scoring techniques.

# Objectives
- Build a structured dataset from raw web data
- Normalize laptop specifications for fair comparison
- Create a weighted scoring system for ranking
- Enable user-driven decision-making

# Data Pipeline
Data Extraction
- Imported web data using PowerQuery
- Data Transformation
- Cleaned and formatted raw datasets
- Applied programmatic transformations when needed
- Standardized inconsistent values
Data Modeling
- Combined:
  - Laptop specifications dataset
  - Benchmark dataset
  - Created relationships between datasets
  - Applied feature engineering and normalization

# Key Features
- Weighted scoring system for ranking laptops
- User profile-based prioritization
- Dynamic filtering and ranking
- Interactive dashboard with slicers

# Tools & Technologies
- Microsoft Excel
- PowerQuery (ETL)
- PowerPivot
- DAX 
- Conditional Formatting

# Skills Demonstrated
- Data Sourcing
- Data Cleaning
- Data Modeling
- Feature Engineering
- Data Normalization
- Dashboard Development

 # Dataset
- Raw laptop specifications from [source](https://www.kaggle.com/datasets/ankit07chy/laptopuncleaneddataset)
- Benchmark dataset for performance comparison from [source](https://browser.geekbench.com)

# How to Use
- Refresh data
- Select user profile (will resolve to default if all is selected)
- Use filters to refine laptop selection
- View ranked results dynamically
